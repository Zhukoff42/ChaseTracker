# 🚔 ChaseTracker - Ваш надежный помощник в погонях!
<img width="1366" height="768" alt="03 32 16 328" src="https://github.com/user-attachments/assets/a027d337-af68-4cd2-9ef5-8db68d801665" />

ChaseTracker - это удобный скрипт для автоматизации работы сотрудников МЮ, который поможет не упустить цель из виду. 🚀

### ✨ Основные возможности:
*   **HUD погони:** Отображение ника, ID, статуса цели и таймера `/z` прямо на экране. 📊
*   **Маркер над целью:** Автоматическая стрелочка над головой подозреваемого, чтобы вы никогда не потеряли его в толпе! 🎯
*   **Системные бинды:** Быстрый доступ к командам `/z`, `/cuff`, `/pull`, `/gotome` и `/domkrat` в один клик. ⚡
*   **Свои команды:** Возможность добавлять собственные бинды через удобное меню. 🛠️
*   **Гибкая настройка:** Полностью настраиваемый интерфейс: меняйте цвета, размеры и положение элементов под себя! 🎨
*   **Автообновление:** Скрипт сам сообщит, когда выйдет новая версия. 🔄

### ⚙️ Установка:
1. Убедитесь, что у вас установлен [MoonLoader](https://www.blast.hk/threads/13305/).
2. Переместите файл `ChaseTracker.lua` в папку `moonloader`.
3. Заходите в игру и используйте команду `/chase` для открытия меню настроек.

### 📝 Управление:
*   `/chase` - открыть меню настроек.
*   `/trg [ID]` - назначить цель вручную.
*   Двойное нажатие **ПКМ** по игроку - быстрый таргет.
